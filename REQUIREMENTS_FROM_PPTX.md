# 자막 UI/UX 요구사항 (자막ui디자인자인_260519.pptx)

## 📋 핵심 요구사항

### 1. 전체 레이아웃 (4개 블록)

**① 상단 옵션 블록** - 메뉴바 + 툴바 (화면 상단)

**나머지 블록 배치:**
```
┌─────────────────┬─────────────────┐
│ ②자막 수정 블록  │ ③플레이어 블록   │
│ (텍스트 편집기)  │ (비디오)        │
├─────────────────┼─────────────────┤
│ ④파형 블록      │  자막 리스트     │
│ (파형 뷰어)     │ (테이블)        │
└─────────────────┴─────────────────┘
```

**✅ 각 블록은 크기 조정이 가능해야 함** - 완료!

※ ①②③④는 설명 순서이며 사분면 위치가 아님

### 2. 디자인 원칙
- ✅ SubtitleEdit과 최대한 유사하게 구성
- ✅ 화이트 모드 (다크 모드 아님 - 작업자들이 선호)
- ✅ 그래픽 아이콘, 폰트 등 동일하게

### 3. 상단 메뉴 변경사항

#### 파일 메뉴
1. **"처음으로"** → **"처음 작업 상태로"**
   - AI 엔진이 자막 추출한 초안 상태로 되돌림

2. **"업로드"** → **"자막 열기"**
   - 외부 자막 파일을 가져와서 편집

3. **"임시파일 불러오기"** 추가
   - 작업 중에 저장되는 자막들 불러오기

4. **"다른 이름으로 저장"**
   - 오프라인으로 저장

#### 편집 메뉴 (모두 유지)
- ✅ 실행 취소 (Undo)
- ✅ 다시 실행 (Redo)
- ✅ 히스토리 보기 (되돌리기용)
- ✅ 찾기
- ✅ 다음 찾기
- ✅ 바꾸기
- ✅ 다중 바꾸기
- ✅ 자막 번호로 이동
- ✅ 선택해서 바꾸기
- ✅ 선택 반전
- ✅ 모두 선택

#### 도구 메뉴
**일반적인 오류 수정 (Ctrl+Shift+F)** - 체크리스트 제공:
- [ ] 빈 자막 제거
- [ ] 불필요한 공백 제거
- [ ] 줄 바꿈 문제 수정
- [ ] 청각 장애인용 텍스트 제거
- [ ] 대/소문자 수정
- [ ] 구두점 수정
- [ ] 시간 코드 정리
- [ ] 중복 자막 제거
- [ ] 간격 조정
- [ ] 기타...

**마무리 작업 도구** (작업자들이 자주 사용):
- [ ] Netflix 품질 검사
- [ ] 시간 코드 정리
- [ ] 읽기 속도 검사
- [ ] 자막 통계

### 4. 자막 리스트 기능

#### 클릭 기능
- ✅ **클릭 시**: 플레이어와 연동 및 수정 가능 모드
- 선택한 자막의 시작 시간으로 비디오 이동

#### 우클릭 메뉴 (컨텍스트 메뉴)
**라인 우클릭 시:**
- 잘라내기
- 복사
- 붙여넣기
- 삭제
- 분할
- 병합
- 시간 조정
- 기타...

**빈 공간 우클릭 시:**
- 새 자막 추가
- 모두 선택
- 붙여넣기

### 5. 플레이어 기능
**✅ 모든 기본 기능 동일하게 동작:**
- ✅ 재생/일시정지
- ✅ 배속 조절
- ✅ 영상 클릭 시 재생/멈춤
- ✅ 프레임 단위 이동
- ✅ 키보드 단축키
- 자막 오버레이

### 6. 파형 블록 기능

**✅ 필수 기능:**
- ✅ 파형 표시
- ✅ 자막 블록 표시
- ✅ 재생 위치 표시
- ✅ 클릭으로 탐색

**우클릭 메뉴:**
- 줌 인/아웃
- 자막 추가
- 자막 이동
- 선택 영역 재생

**빈 공간 드래그:**
- 새 자막 생성
- 시간 범위 선택

### 7. 키보드 단축키

**자막 편집:**
- `Q` - 이전과 병합
- `W` - 다음과 병합
- `A` - 다음 줄로 이동
- `F` - 이전 줄로 이동
- `S` - 시작 시간 설정
- `D` - 적절한 줄에 대한 끝 설정
- `Ctrl+Shift+Q` - 자막 사이의 간격 메우기
- `Ctrl+Shift+F` - 일반적인 오류 수정

**기타:**
- `Ctrl+Z` - 실행 취소
- `Ctrl+Y` - 다시 실행
- `Ctrl+F` - 찾기
- `Ctrl+H` - 바꾸기
- `Ctrl+G` - 자막 번호로 이동
- `Space` - 재생/일시정지
- `←/→` - 프레임 이동

### 9. 추가 기능
**자막 생성 엔진 설정:**
